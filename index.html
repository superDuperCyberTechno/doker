Doker is a super simple, betting game that uses dominoes, dice and chips.

What you need

  • 1 double-six standard set of dominoes
  • 2 standard six-sided dice.
  • A lot of chips.

Gameplay

0. Setup
Decide on a dealer.
Place all dominoes face down on the table.
Every player is given 3 dominoes from the domino pool.

1. First roll
The dealer adds one chip to the pot and rolls one die.

2. First bet
Every player decides (see Evaluating dominoes) if they want to bet on the outcome - in a clockwise fashion, starting with the dealer.
If a player wants to bet, he/she plays their chosen domino face down on the table and the dealer adds a chip the pot. The betting round ends when it loops around to the dealer.

3. First check
If only one player participated in the first bet, he/she wins the round and you go directly to 7.

4. Second roll
The dealer rolls the second die.

5. Second bet
Every player with an active bet decides - in a clockwise fashion, starting with the dealer - if they still want to continue.
For every continuing player, the dealer adds one chip to the pot.

6. Second check
If only one player participated in the second bet, he/she wins the round and you go directly to 7.
Every participating player - in a clockwise fashion, starting with the dealer - flips their played domino. The player with the highest valued domino (see Evaluating dominoes) wins the round.

7. Conclusion
The winner is awarded all chips in the pot.
All played dominoes are added back to the domino pool.
If any player has no dominoes left, he/she can buy more from the domino pool at 1 chip per domino. Any chips spent this way is added to the next round's pot. Players cannot have more than 3 dominoes in their hand at any time.
Shuffle the domino pool.
The player on the dealer's left is the next round's dealer.
Restart the round (go to 1).

End
The game can be played indefinitely, on a timer or until a set chip sum has been reached.

Evaluating dominoes

The main goal is to play the highest graded domino when compared to the 2 dice. The dice dictates what values the dominoes need to consist of. The 2 values on the dominoes match one dice, respectively. - Thus, both domino values are always evaluated. To evaluate a domino grading, do the following:
  1. Check if there are any hard matches (2 hard matches always win the bet). A hard match is any number that exist on both domino and dice.
  2. Within any remaining domino values, determine soft matches. A soft match is any blank domino value, this always matches to the highest non-matched value.
