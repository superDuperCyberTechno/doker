A decidedly uninformed remake of poker
Version 0.9

The original author of this game likes poker. But he does not know how to play it, neither does he intend to learn. Instead he chose to cram a bunch of assumptions about the poker into a whole new boardgame.
The vision was simple: It must be easy to assemble, learn and play.
The result is a game that doesn't use a deck of cards. Instead it uses dice and dominoes to (probably) create a refined and easier version of poker.

Doker is open source and contributions are very welcome.

What you need

1 double-six standard set of dominoes (28 pieces).
2 standard six-sided dice.
A lot of chips (or any other token).
2-9 players.

Gameplay

0. Setup
Decide on a dealer. Note: The dealer is also an active player exactly as any other participant.
Place all dominoes face down on the table. - This is your domino pool.
Every player receives 3 dominoes.
Decide on a chip pool size. Default is 25.

1. First roll
The dealer rolls the first die.

2. First bet
A chip is added to the pot.
Everyone decides (including the dealer) if they want to participate - in a clockwise fashion, starting with the dealer. For any player that wants to participate, he/she plays their chosen domino face down on the table and a chip is added to the pot. Everyone is free to participate if there's not enough chips to add to the pot.
Any player that decides not to participate is excluded from the rest of the round.

3. First check
If no one participated in the first bet, you go directly to 7.
If only one player participated in the first bet, he/she wins the round and you go directly to 7.

4. Second roll
The dealer rolls the second die.

5. Second bet
Every participant decides (see Evaluating dominoes) if they still want to participate - in a clockwise fashion, starting with the dealer. Every participating player adds a chip to the pot. Everyone is free to participate if there's not enough chips to add to the pot.
Any player that decides not to continue is excluded from the rest of the round.

6. Second check
If no one participated in the second bet, you go directly to 7.
If only one player participated in the second bet, he/she wins the round and you go directly to 7.
Every participating player - in a clockwise fashion, starting with the dealer - flips their played domino. The player with the highest valued domino (see Evaluating dominoes) wins the round.

7. Conclusion
If a winner is found, they are awarded all chips in the pot.
All played dominoes are added back to the domino pool. The domino pool is then shuffled.
If any player has no dominoes left, he/she can buy more from the domino pool at 1 chip per domino. Any chips spent this way is added to the next round's pot. Players cannot have more than 3 dominoes in their hand at any time.
The player on the dealer's left is the next round's dealer.
Restart the round (go to 1).

Evaluating dominoes

UNDER CONSTRUCTION

The main goal is to play the highest graded domino when compared to the 2 dice. The dice dictates what values the dominoes need to consist of. The 2 values on the dominoes match one dice, respectively. - Thus, both domino values are always evaluated.

To evaluate a domino grading, do the following:
  1. Check if there are any hard matches (2 hard matches always win the bet). A hard match is any number that exist on both domino and dice.
  2. Within any remaining domino values, determine soft matches. A soft match is any blank domino value, this always matches to the highest non-matched value.
Variants

Cash mode
If you're the kind of maniac that likes to play with real money, this mode is for you. Basically, all chips have a fixed monetary value and are supplied by players themselves.
On first roll, the chip is supplied by the dealer.
On first, - and second bet, all chips are supplied by the betting player.
At the end of the conclusion, players are allowed to purchase new chips with their own money.
If if player has no chips, he/she cannot be dealer or participate in bets.
